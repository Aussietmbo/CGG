# Cellular Automata

## Simulators

### [Coherence-Driven Cellular Automata v4.2](coherence-driven-ca_fft_v4_2.html)
Latest version of coherence-driven cellular automaton with FFT-based spectral feedback.

**Features:**
- 2D/3D grid with periodic boundary conditions
- FFT-based spectral feedback loop
- Coherence coupling dynamics
- Parameter sweep capabilities
- Automatic CSV data export

**Technical Specifications:**
- FFT Decomposition for spectral analysis
- Mulberry32 PRNG for reproducibility
- Tuned parameters for balanced growth/decay
- Validated against physics models

**Use Cases:**
- Phase diagram exploration (diffusion sweep)
- Glider detection and dynamics measurement
- Spatial autocorrelation analysis
- Parameter sensitivity studies
