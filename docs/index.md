## 📊 Featured Simulators

### TRR-G Vortex Nucleation
2D interactive exploration of vortex formation in compact object interiors with echo physics.

### TRR Wave Tank (GPU)
GPU-accelerated wave propagation with real-time coherence dynamics visualization.

### Coherence-Driven Cellular Automata
Large-scale parameter sweep tool for exploring phase diagrams and glider detection.

### TRR-G Light Propagation
Ray optics in scalar field backgrounds with analytical validation.
